Why Use Default Export?

In JavaScript ES6, the default keyword is used to export a single entity from a module, designating it as the default export.

What is Default Export?

What is Default Export? Using export default allows you to export a single main item from a module. For example, in React, the component itself is often exported as the default export.

export default MyComponent; import MyComponent from './MyComponent';

This syntax simplifies the import because you don't need to match the exported name with the import statement.

Why Use Capital Letters for Components?

In JavaScript, specifically in frameworks like React, it's a convention to use capital letters when defining components. This helps distinguish components from regular HTML elements.
